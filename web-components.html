input组件

input组件包括一般的输入框以及时间和日期区间，input组件占据100%宽度，两端宽度自适应

							
								<div class="ysh-demo">
								<div class="input-group">
									<label class="input-label-base">商品编码/条码/名称/物流码</label>
									<input type="text" placeholder="20151012545786584458" class="input-base">
								</div>
								</div>

表格

默认的带有.table类的表格行内是没有边框的，若需加边框，需加.table-border类。

分页

分页中当前页加类.active,首页与末页分别加.prev、.next，最后的词条加total类。

  • 首页
  • 1
  • 2
  • 30
  • 末页
  • 1-10条，共100条 
							
								<div class="ysh-demo">
									<div class="pagination">
										<ul>
											<li class="prev">
												<a href="#">首页</a>
											</li>
											<li>
												<a href="#">1</a>
											</li>
											<li class="active">
												<a href="#">2</a>
											</li>
											<li>
												<a href="#">30</a>
											</li>
											<li class="next">
												<a href="#">末页</a>
											</li>
											<li class="total">
												1-10条，共100条
											</li>
										</ul>
									</div>
								</div>
