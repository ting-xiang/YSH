HTML/CSS编写规范

YSH-UI的css是经过less编译而来的。css的基础命名方式为{模块名}+{状态}，通过css的命名，大概知道是干什么的。如下：

  • .dropdown{ }
  • .dropdown-icon{ }
  • .button-orange-small{ }

JS编写规范

与JS相关的css我们使用钩子方式，即给js加前缀J-，且该css不写任何样式。如下：

  • .J-dropdown{ }
  • .J-return-top{ }

重置样式库

为了增强css样式跨游览器表现的一致性，使用了css统一样式库normalize

目录结构

  • base.less ——less变量，定义基础颜色，字体，边框，边距等等。
  • layout.less ——自定义的其他样式
  • button.less、dialog.less、inpu.less等 ——自定义的组件
