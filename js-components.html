确认取消框

这是一个自定义标题和按钮的确认取消弹出框，在html中引入.ysh-dialog类， 引入ysh.js，调用如下： 

							
							     $("#trigger-btn").on("click",function(){   //#trigger-btn 触发事件的按钮
						            $.dialog({
						                title: '确认删除吗!',  //提示内容
						                buttons   : {
						                    confirm   : {'name' : '确定', 'action': function(){alert("ddd");}},   //按钮名
						                    cancel    : {'name' : '取消','action': function(){console.log(25);}}
						                }
						            });
						   	 	});
